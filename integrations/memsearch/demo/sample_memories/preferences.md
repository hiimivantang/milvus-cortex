# User Preferences

## Editor Setup

I prefer VS Code with the Vim extension for all my coding. My theme is Catppuccin Mocha
and I use JetBrains Mono as my font. I keep my terminal in the integrated panel and
use tmux for long-running processes.

## Communication Style

I like concise answers without unnecessary preamble. When explaining code, show me the
diff rather than describing changes in prose. I prefer bullet points over paragraphs.

## Tech Stack Preferences

For new Python projects I always start with:
- uv for package management (not pip or poetry)
- ruff for linting and formatting
- pytest for testing
- pydantic for data models

For databases, I prefer PostgreSQL for relational and Milvus for vector search.
I use Docker Compose for local development environments.
